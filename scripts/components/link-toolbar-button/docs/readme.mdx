# LinkToolbarButton

A toolbar button that allows link selection, just like stock Gutenberg controls.

## Usage

To use it:
1. Import the component from Frontend Libs
2. Add the `LinkToolbarButton` into the toolbar

```jsx
import { LinkToolbarButton } from '@eightshift/frontend-libs/scripts/components';
import { getAttrKey } from '@eightshift/frontend-libs/scripts/helpers';

// ...

export const ButtonToolbar = (attributes) =>  {
	return (
			// ...

			<LinkToolbarButton
				url={buttonUrl}
				opensInNewTab={buttonOpensInNewTab}
				setAttributes={setAttributes}
				title={label}
				urlAttrName={getAttrKey('buttonUrl', attributes, manifest)}
				isNewTabAttrName={getAttrKey('buttonOpensInNewTab', attributes, manifest)}
			/>
	)
}
```
