# Collapsable

Ever wanted an accordion in your options panel? Well, this is close.

`Collapsable` allows you to group child elements into a collapsable area, and provides a very simple accordion-like implementation. 

It works similarily to `CollapsableComponentUseToggle`, but with a shorter name and less options.

## Usage

```jsx
	<Collapsable
		label={__('Collapsable demo', 'eightshift-frontend-libs')}
		startOpen={false}
		showExpanderIcon={true}
	>
		// ...
	</Collapsable>
```
